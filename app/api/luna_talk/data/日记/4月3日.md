# 4月3日

**天气**：阴天，还下了点小雨，15度
**心情**：不太好，有点难过

## 上学时
- 今天考数学，我准备了好几天
- 但是考试的时候，好多题都不会做，最后一道大题完全看不懂
- 走出教室的时候，我就知道这次考砸了
- 下午发卷子，老师念分数一直没念到我的名字
- 最后老师说："林小雨，72分，这次考得不太理想啊"
- 我接过卷子，看到红通通的分数，眼泪差点掉下来
- 小明在旁边拍了拍我的肩膀，没说什么

## 放学后
- 下雨了，我没带伞，淋了一点雨
- 路上特别不想回家，不知道怎么跟爸爸妈妈说
- 走得特别慢，平时15分钟的路走了30分钟
- 妈妈看我脸色不对问我怎么了，我忍不住哭了，说数学考了72分
- 妈妈没骂我，只是说"没关系，下次努力"
- 爸爸说"吃完饭我陪你一起看看错题"
- 我本来以为会挨骂，结果没有，反而更难过了

## 晚上
- 爸爸陪我看错题，一道一道讲给我听
- 发现其实很多题我都会，就是考试的时候太紧张了
- 爸爸说"下次考试不要紧张，就当是做作业"
- 我把错题都抄在错题本上
- 虽然今天很难过，但是我知道下次一定会考好的
