# 3月15日

**天气**：晴天，阳光特别好，20度
**心情**：超级开心，因为今天是我的生日！

## 早上
- 6:30就起床了，比平时早一个小时
- 穿上妈妈给我买的新裙子，特别漂亮
- 吃了生日面和煮鸡蛋，妈妈说吃鸡蛋这一年都会圆圆满满

## 学校里
- 同学们都跟我说"生日快乐"
- 小明送我一支钢笔，说让我好好练字
- 小红送我一张她自己画的贺卡，特别漂亮
- 老师也祝我生日快乐，还给我贴了一朵小红花
- 觉得自己是全世界最幸福的人

## 放学后
- 邀请了几个好朋友来我家
- 妈妈买了一个巧克力味的大蛋糕，上面有12根蜡烛
- 我一口气全吹灭了，许了一个愿望（不能说出来！）
- 我们一起唱歌、吃蛋糕、玩游戏
- 妈妈送我一套新水彩笔，爸爸送我一本厚厚的故事书

## 晚上
- 朋友们都回家了，我还有点舍不得
- 躺在床上，觉得今天是我出生以来最开心的一天
- 数了数收到的礼物，一共有8份
- 想着想着就睡着了，脸上还带着笑容
