# 我的学校

我的学校叫阳光小学，建于2010年。

## 校园建筑
- **教学楼**：红色砖墙4层，每层6间教室，走廊有同学的画
- **操场**：红色200米跑道，绿色草坪，2个篮球场，4张乒乓球台
- **图书馆**：一楼东侧，约5000本书，有动物故事区和漫画区
- **花园**：教学楼和操场之间，有迎春、月季、菊花、梅花，中间有小石亭
- **美术教室**：三楼西侧，有画架和水彩颜料，墙上挂满同学的画
- **音乐教室**：美术教室旁边，有钢琴和打击乐器，合唱团在这里排练
- **科学实验室**：四楼，8张实验台，有显微镜、烧杯等器材
- **体育馆**：操场北边独立建筑，室内篮球场和羽毛球场
- **食堂**：西北角，11:30-12:30营业，糖醋排骨、番茄炒蛋很好吃
- **校门**：南边，黑色铁门，有金色"阳光小学"字样
