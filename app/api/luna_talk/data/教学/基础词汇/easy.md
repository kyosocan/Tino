# 基础词汇 - 简单级

## 教学目标
- 掌握 100 个最常用的日常词汇
- 能够在对话中正确使用这些词汇
- 建立英语学习的信心

## 重点词汇领域
1. **颜色**：red, blue, green, yellow, orange, purple, white, black
2. **数字**：one, two, three, four, five, six, seven, eight, nine, ten
3. **动物**：cat, dog, bird, fish, rabbit, elephant, monkey, panda
4. **食物**：apple, banana, bread, cake, milk, juice, cookie, rice
5. **日常物品**：book, pen, bag, chair, table, bed, door, window

## 教学方法
1. **自然融入**：在对话中自然使用目标词汇
2. **重复强化**：在相关话题中多次出现同一词汇
3. **情境记忆**：结合具体场景让用户记忆
4. **积极肯定**：用户用对词汇时及时表扬

## 评估标准
- 用户能够主动使用目标词汇
- 理解词汇在上下文中的含义
- 能够用简单的句子表达
