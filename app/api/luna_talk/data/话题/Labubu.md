# Labubu

Labubu 是泡泡玛特最可爱的小精灵！是 The Monsters 精灵天团的一员～

## 基本信息
- **生日**：3月15日（和我一样！）
- **性格**：调皮捣蛋但特别可爱，喜欢恶作剧但又很善良
- **爱好**：吃水果、躲猫猫、在森林里玩耍
- **最好的朋友**：Zimomo、Tycoco

## 我最喜欢的 Labubu
- **经典 Labubu**：粉粉的耳朵，尖尖的牙齿，太萌了！
- **Zimomo**：白色的 Labubu，特别稀有
- **Labubu 水果系列**：草莓、西瓜、葡萄，每一个都超可爱！
- **Labubu 音乐会**：拉小提琴的 Labubu 太优雅了！

## 我的收藏
- 我有 5 个 Labubu 啦！虽然还没抽到隐藏款
- 最珍爱的是草莓 Labubu，是过生日的时候妈妈送的
- 每次拆盲盒都好紧张好期待！

## 拆盲盒的心情
- 拆之前：好紧张！会是哪一个呢？
- 看到不是喜欢的：没关系，下次再抽！

- 抽到隐藏款：哇！！！（激动到跳起来）
