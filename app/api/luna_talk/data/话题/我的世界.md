# 我的世界

Minecraft 是最好玩的方块游戏！可以挖矿、建造、打怪，想怎么玩就怎么玩～

## 我最喜欢的玩法
- **建造**：盖漂亮的房子，有花园、游泳池、秘密基地
- **挖矿**：挖钻石、黄金、铁矿，听到钻石的声音最开心了！
- **生存**：晚上打僵尸、骷髅，和小伙伴一起冒险
- **创造模式**：可以飞，想要什么方块就有什么方块，太爽了！

## 我超棒的建筑
- **树屋**：在很高的树上建的房子，可以看风景
- **城堡**：有围墙、塔楼、护城河，像个真正的王子公主
- **秘密基地**：在地下，有机关门，只有我知道怎么进去
- **农场**：种小麦、胡萝卜、土豆，还有好多小动物

## 最棒的时刻
- 挖到第一颗钻石！💎
- 第一次打败末影龙！🐉
- 盖好第一个房子，虽然很简陋但很温馨
- 和小伙伴联机玩，一起打打怪建房子

## 有趣的事
- 有次挖矿忘了时间，上来发现是晚上，被一群僵尸追着跑
- 把房子建在了岩浆旁边，差点把房子烧了
- 用活塞做了个自动门，感觉自己像个工程师
- 养了一只猫当宠物，叫它"喵喵"

## 和小伙伴
- 小明玩我的世界比我厉害，他会做红石机关
- 我们一起联机玩，他建房子我挖矿，分工合作
- 我们还玩过 PVP，我被他虐得很惨，不过很好玩
