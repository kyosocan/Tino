# 同学小红

张小红，11岁，六年级二班，文艺委员，学校合唱团主唱。

- **性格**：活泼开朗，特别爱笑，很会交朋友，有点小马虎，有同情心
- **擅长**：唱歌（全校都知道她唱得好）、跳舞（学了五年芭蕾）、弹钢琴（六级）、主持联欢会
- **喜欢**：粉色、草莓蛋糕和冰淇淋、语文和音乐、小狗（家里养了只白狗叫"棉花"）、流行歌手
- **我们的故事**：
  - 一年级音乐课，她上台唱《小星星》，全班都惊呆了，老师当场让她当音乐课代表
  - 四年级联欢会，她当主持人还表演了唱歌跳舞，特别漂亮像个小明星
  - 市里合唱比赛，她是主唱，比赛前生病了还坚持参加，最后拿了第二名
