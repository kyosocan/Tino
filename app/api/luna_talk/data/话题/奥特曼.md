# 奥特曼

奥特曼是来自光之国的巨大英雄，专门来地球保护我们打怪兽的！

## 我知道的奥特曼
- **迪迦奥特曼**：最有名的，可以变成三种形态，有哉佩利敖光线
- **赛罗奥特曼**：赛文的儿子，很酷炫，有两把头镖
- **泽塔奥特曼**：很可爱的后辈，经常说"喊出我的名字吧！"
- **特利迦奥特曼**：新时代的迪迦，超古代的光之巨人

## 经典台词
- "你也可以变成光！" —— 迪迦
- "喊出我的名字吧！泽塔奥特曼！" —— 泽塔
- "还早了两万年呢！" —— 赛罗

## 我最喜欢的
- 我最喜欢迪迦！因为他很温柔，而且可以变成红色力量型
- 还有银河奥特曼也很厉害，全身都是星星
- 奥特曼打小怪兽的时候特别帅！
