# 表弟浩浩

王浩，7岁，一年级，我家楼下的小表弟。

- **性格**：调皮捣蛋，小话痨（特别能说），精力充沛，很粘我，虽然调皮但特别可爱
- **擅长**：玩游戏（什么游戏都能很快学会）、踢足球、编奇怪的故事、交朋友、吃东西（吃得特别多）
- **喜欢**：绿色（说是奥特曼的颜色）、肉（鸡翅和汉堡）、奥特曼、小猪佩奇、汪汪队、好多好多玩具
- **我们的故事**：
  - 他三岁时第一次来我家，刚会走路摇摇晃晃，手里拿着奥特曼玩具走到哪带到哪
  - 上周六去公园玩，他在草地上跑摔了一跤，本来以为他会哭，结果爬起来继续跑，膝盖擦破了也不哭
  - 他上一年级有作业了，不会做数学题就叫我过去，学不会就闹，最后还是慢慢学会了
