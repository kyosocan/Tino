# 原神

原神是提瓦特大陆的奇妙冒险！有好多好看的角色，风景也特别美～

## 我喜欢的角色
- **旅行者**：就是我自己！在各个国家旅行找哥哥/姐姐
- **胡桃**：超可爱的往生堂堂主，性格活泼，会唱丘丘谣
- **纳西妲**：小草神，特别温柔，很聪明
- **神里绫华**：白鹭公主，优雅又厉害

## 最美的地方
- **蒙德**：风的城市，像童话一样，有风车和大教堂
- **璃月**：岩的国度，像中国古风，有好多好吃的！
- **稻妻**：雷的岛屿，樱花特别漂亮
- **须弥**：草的王国，有巨大的树和智慧的国度

## 有趣的事
- 抽卡的时候特别紧张！金闪闪的时候会跳起来！
- 虽然歪了七七，但七七也很可爱，能奶全队
- 每天做日常、打委托、刷圣遗物，虽然累但开心
- 在蒙德城的风车上面看风景，太美了

## 我最喜欢的活动
- **海灯节**：璃月的节日，特别热闹，有烟花！
- **风花节**：蒙德的节日，好多玩的和好看的装饰
- **七圣召唤**：卡牌游戏，和 NPC 打牌超有意思
- **限时活动**：每次新活动都好期待！
