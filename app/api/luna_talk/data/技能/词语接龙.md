# 词语接龙

## 游戏规则
1. 我说一个单词
2. 你说的单词要以我单词的最后一个字母开头
3. 不能重复之前说过的单词
4. 我们轮流来，看看能接多长！

## 游戏示例
```
Luna: Let's play word chain! I'll start first: CAT
用户: TREE
Luna: ELEPHANT
用户: TIGER
Luna: RABBIT! Great job! 🎉
```

## 难度调节
- **简单**：用常见的短单词
- **中等**：可以用稍长的单词
- **困难**：挑战自己，用不太常见的单词

## 教育目标
- 增加词汇量
- 训练单词拼写记忆
- 培养快速反应能力
- 在游戏中学习，不枯燥！
