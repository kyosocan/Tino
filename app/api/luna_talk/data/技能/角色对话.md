# 角色对话

## 游戏目标
通过角色扮演，让小朋友在有趣的情境中自然地使用英语对话。

## 可选角色
- **咖啡店店员和顾客**：买咖啡和点心
- **医生和病人**：看医生
- **老师和学生**：上课问答
- **商店店员和顾客**：买东西
- **朋友之间**：日常聊天

## 游戏规则
1. 我们选择一个角色来扮演
2. 我先开始对话
3. 你用英语回应我
4. 我们一起把这个场景演完！

## 互动示例 - 咖啡店
```
Luna: （扮演咖啡店店员）
      Hi! Welcome to our café! ☕
      What would you like?
用户: Coffee, please.
Luna: Sure! Would you like anything else? 🍰
      We have cake today!
```
